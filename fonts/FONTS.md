# Fonts

Place these files here to activate Chicago FLF:

1. Download from: https://www.fontsquirrel.com/fonts/chicago-flf
2. Extract the archive
3. Copy `ChicagoFLF.ttf` into this folder
4. Optionally convert to `.woff2` for faster loading (use https://cloudconvert.com/ttf-to-woff2)

The site falls back to Space Mono if these files are absent.
